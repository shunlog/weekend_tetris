# Tetris in a weekend

# Features

- Following the [Tetris guideline](https://tetris.wiki/Tetris_Guideline)
- Wall kicks, [Super Rotation System](https://tetris.wiki/Super_Rotation_System) 
- Delayed auto-shift (DAS)
- Soft drop
- Lock delay
